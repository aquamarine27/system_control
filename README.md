# system_control


---

##  Tech Stack

- **Backend:** [Go](https://go.dev/) with [Fiber] & [Gorm]
- **Frontend:** [React](https://react.dev/)  
- **Database:** [PostgreSQL](https://www.postgresql.org/)  


---

## Getting Started



### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/aquamarine27/system_control.git
   cd system_control
   ```
2. **Build frontend**
   ```bash
   cd frontend
   npm install
   npm run dev
   ```
3. **Build backend**
   ```bash
   cd backend
   go mod tidy
   go run main.go
   ```
